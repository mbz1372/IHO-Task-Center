# IranHotel OS V9 — Online Operations

## تغییرات
- تعداد کل هتل‌ها مستقیماً با `count exact` از `ihos_hotels` خوانده می‌شود.
- تعداد تخت فقط در دیتابیس نگهداری می‌شود و از رابط سوپر اپ حذف شده است.
- تمرکز رابط بر منبع نرخ، منبع ظرفیت و آنلاین بودن واقعی است.
- دکمه «ایجاد تسک» در جدول و پرونده هر هتل اضافه شده است.
- تب تحلیل کارشناسان برای IHO Provider اضافه شده است.
- Task ID 1 = کارشناس ظرفیت؛ Task ID 2 = کارشناس نرخ.
- حذف تکی فقط تأیید سوپر ادمین می‌خواهد.
- فقط حذف کامل داده‌ها رمز می‌خواهد.
- رمز حذف کامل از تنظیمات سیستم و فقط توسط سوپر ادمین قابل تغییر است.

## راه‌اندازی
1. فایل `SUPABASE-V9-ONLINE-EXPERTS.sql` را در SQL Editor اجرا کنید.
2. متغیر `SUPABASE_SERVICE_ROLE_KEY` باید در Vercel موجود باشد.
3. پس از Deploy با سوپر ادمین وارد شوید و از تنظیمات، رمز حذف کامل را تعیین کنید.

## کنترل فنی
- `npm run build` موفق
- `npx tsc --noEmit` موفق
