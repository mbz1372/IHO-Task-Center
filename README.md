# IranHotel Operations System Enterprise V4

نسخه Enterprise V4 با تمرکز روی عملیات واقعی تیم تامین ایران‌هتل.

## امکانات اصلی

- Task Center با تغییر مسئول، وضعیت، برچسب‌ها، همکاران مشارکت‌کننده و Progress Bar فعالیت‌ها
- Roles & Permissions برای کنترل نمایش منوها و اکشن‌ها
- مدیریت کامل کاربران و افزودن کاربر جدید
- یادآور با تاریخ و ساعت، همراه با Chrome Notification
- تب لاگ اتفاق‌ها با نمودار فعالیت کاربران و خروجی CSV
- فعالیت‌های داخل هر تسک، زمان تخمینی، وضعیت انجام و زمان انجام
- پین کردن یادداشت مهم در تسک‌ها، پروژه‌ها و اسناد
- هدف‌گذاری روی فعالیت‌ها و تسک‌های انجام‌شده
- پروژه‌ها با امکان مشارکت چند کاربر
- اتوماسیون ساده برای اعمال خودکار مسئول، اولویت، وضعیت و برچسب هنگام ایجاد تسک
- تقویم ماهانه شمسی با نمایش تسک‌ها، رویدادها و یادآورها
- ورود گروهی هتل‌ها از Excel مطابق ستون‌های فایل ایران‌هتل
- آپلود قرارداد و اسناد روی Supabase Storage
- تنظیمات کامل برندینگ، رنگ، لوگو، favicon، تم، دسته‌بندی‌ها، وضعیت‌ها و برچسب‌ها

## راه‌اندازی

1. محتوای پروژه را در GitHub جایگزین کن.
2. در Supabase → SQL Editor فایل `src/app/schema.sql` را اجرا کن.
3. در Vercel Environment Variables این دو مقدار را داشته باش:
   - `NEXT_PUBLIC_SUPABASE_URL`
   - `NEXT_PUBLIC_SUPABASE_ANON_KEY`
4. در Vercel گزینه Redeploy بدون Cache را بزن.
5. ورود اولیه:
   - username: `admin`
   - password: `123456`

## نکته امنیتی

این نسخه برای استفاده داخلی MVP/Enterprise داخلی با RLS باز طراحی شده است تا سریع عملیاتی شود. برای Production سخت‌گیرانه‌تر، باید Supabase Auth و Policyهای مبتنی بر JWT اضافه شود.
