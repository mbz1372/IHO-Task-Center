# IranHotel OS V10 — Master Hotel Data Fix

## اصلاحات
- صفحه «پرونده هتل‌ها» دیگر از آرایه ۳۰۰تایی Task Manager استفاده نمی‌کند.
- هتل‌ها مستقیماً و صفحه‌بندی‌شده از جدول اصلی `ihos_hotels` خوانده می‌شوند.
- شمارش کل از `count: exact` همان جدول اصلی انجام می‌شود.
- اطلاعات آنلاین‌سازی هر صفحه جداگانه از `ihos_hotel_automation` دریافت و روی هتل‌های همان صفحه Merge می‌شود.
- حتی اگر View تحلیلی ناقص یا قدیمی باشد، هتل Import‌شده در پرونده نمایش داده می‌شود.
- یادآورها توسط همه کاربران دارای دسترسی ماژول یادآورها قابل حذف هستند.
- محدودیت سوپرادمین فقط برای دیتابیس اصلی هتل‌ها، اطلاعات اتوماسیون هتل و Provider Matrix باقی مانده است.
